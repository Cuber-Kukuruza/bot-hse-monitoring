def load_data(): ¶

Загружает данные о подключении к серверам и пороги из файлов.

Восстанавливает ранее сохраненные данные, включая учетные данные серверов и значения порогов, для использования в приложении.

def main(): ¶

Запускает бота.

Инициализирует бота, загружает данные, регистрирует обработчики команд и сообщений, а также запускает периодические задачи. Запускает режим опроса для обработки обновлений.
